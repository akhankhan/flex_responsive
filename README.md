# flex_responsive

A Flutter package that makes responsive design simple and intuitive.

## Features

- 📱 Simple responsive checks (mobile, tablet, desktop)
- 🎯 Adaptive layouts with minimal code
- 📊 Responsive grid system
- 🚀 High performance
- 💪 Zero dependencies
- 🎨 Easy to customize

## Installation

```yaml
dependencies:
  flex_responsive: ^0.0.1
```
